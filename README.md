Printly 3D:

  Printly 3D is a open source web-based 3D printing service platform that allows anyone to browse, upload, and order 3D-printed objects that will be delivered to their house. 
   
About the program:
   
   Built with HTML and CSS, the current version is (v1) and the website is not yet fully loaded up with all the features, it uses deep blue (#3e57d9) background. The page uses fonts that I accessed from fonts.googleapis.com "Questrial" 
   It has a button that links the user to a google form in which the user can submit their 3d model and enter their name with address.
   then the model will be printed using my 3d printers and delivered to their doorstep, payment system yet to be added.
   
   [note: this website is under construction. Project started on 2/april/2026]
   [note: the 3d printing service is only available in india, planning on other coumtries]
